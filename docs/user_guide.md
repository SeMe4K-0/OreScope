# OreScope — руководство пользователя (для геолога)

## 1. Запуск

Дважды кликните `start.bat` (или выполните `python app.py`) и откройте в браузере
`http://localhost:7861`. Всё работает локально — изображения не покидают ваш компьютер.

## 2. Загрузка образцов

1. Перетащите файлы шлифов (TIFF/PNG/JPEG, можно сразу партию) в зону «Загрузить изображения».
2. Если известен масштаб съёмки — укажите **мкм/px** до загрузки: в таблице появятся
   абсолютные площади в мм².
3. Образцы встают в очередь и обрабатываются автоматически один за другим.
   Панорама ~300 Мп занимает около 2 минут, обычный кадр — секунды.

## 3. Чтение результата

**Цвета маски** (кнопка «Маска»):
- 🟢 зелёный — обычные срастания: крупные, слабо замещённые сульфиды → рядовая руда;
- 🔴 красный — тонкие срастания: сульфиды, значительно замещённые нерудной фазой → труднообогатимая;
- 🔵 синий — тальк (тёмная рассеянная фаза в нерудной матрице).

**Карта уверенности** — тепловая карта вероятности талька: синие/красные зоны около
0 или 1 — уверенные решения, жёлто-зелёная середина — спорные участки, посмотрите их глазами.

**Заключение** формируется по правилу: тальк > 10 % → оталькованная; иначе — по
преобладающему типу срастаний. Колёсико мыши — зум, перетаскивание — панорамирование.

## 4. Типичные случаи

| Что видите | Как интерпретировать |
|---|---|
| Крупные зелёные массивы, талька < 2 % | классическая рядовая руда |
| Ажурная красная «решётка» по всей площади | труднообогатимая, тонкое замещение |
| Синие поля > 10 % при любых срастаниях | оталькованная, приоритет талька в правиле |

## 5. Пограничные случаи (система предупредит сама)

- **Тальк 9–11 %** — значение на границе порога при допуске оценки ±3 п.п.;
  вручную проверьте синие зоны, статус образца станет «Проверка».
- **Обычные ≈ тонкие (разница < 0.5 %)** — класс выбран детерминированно
  (равенство трактуется в пользу труднообогатимой), но требуется взгляд эксперта.
- **Сульфидов < 0.1 %** — «пустая порода / не классифицирована»: метрики срастаний
  на единичных зёрнах статистически не значимы.
- **Низкая уверенность модели талька** (тёмные панорамы, нетипичная съёмка) —
  вердикт остаётся, но образец помечается на проверку.

## 6. Экспертная проверка и дообучение

Кнопки «Подтвердить»/«Отклонить» фиксируют ваш вердикт. Отклонённые образцы и
исправленные маски складывайте в `data/corrections/` — команда переобучит модели
командой `python scripts/train_talc.py --resume models/talc_best.pt` (transfer learning
на новые типы руд или другой микроскоп).

## 7. Экспорт

- **PDF** — готовый отчёт с оверлеем, таблицей и заключением;
- **CSV** — метрики для Excel/LIMS;
- **GeoJSON** — контуры зон (тальк, обычные, тонкие) для ГИС (QGIS открывает напрямую;
  Shapefile — через «Экспорт слоя» в QGIS);
- **JSON-лог** — все параметры запуска для воспроизводимости анализа.
